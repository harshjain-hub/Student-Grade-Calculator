# Student Grade Calculator

## Project Overview

The Student Grade Calculator is a beginner-friendly Python program that accepts a student's name and marks, validates the input, calculates the grade, and displays an encouraging message based on the grade.

---

## Objectives

- Learn how to use input()
- Practice if-elif-else statements
- Understand while loops
- Create and use functions
- Validate user input
- Display formatted output

---

## Technologies Used

1. Install Python 3.


## Grading Logic

| Marks | Grade |
|--------|-------|
| 90–100 | A |
| 80–89 | B |
| 70–79 | C |
| 60–69 | D |
| 0–59 | F |
